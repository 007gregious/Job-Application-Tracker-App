# JobTracker - Job Application Tracker

A beautiful and user-friendly job application tracking application that helps you manage your job search journey.

## Features

- 📊 Dashboard with statistics overview
- ➕ Add new job applications with detailed information
- 📋 View and manage all applications
- 🔍 Filter and search applications
- 📱 Responsive design for mobile and desktop
- 📤 Export applications to Excel format
- 💾 Local storage persistence
- 🎨 Modern and intuitive UI

## Technologies Used

- React 18
- React Hooks for state management
- Local Storage for data persistence
- XLSX for Excel export
- date-fns for date manipulation
- react-hot-toast for notifications
- react-icons for icons

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

## Usage

1. **Dashboard**: View statistics and recent applications
2. **Add Application**: Fill in the form with job details
3. **View Applications**: Browse all applications with filters
4. **Export**: Click "Export to Excel" to download your data

## Project Structure

The project follows best practices with:

- Component-based architecture
- Custom hooks for reusable logic
- Service layer for data operations
- Utility functions for common operations
- CSS variables for consistent styling
- Responsive design principles
